## Norrsken - Weather and Northern Lights App
A mobile application built with React Native and Expo that provides location-based weather forecasts, aurora (Northern Lights) predictions, and sunrise/sunset information.

## Overview
This project combines multiple external APIs into a single application, allowing users in northern regions and Scandinavia to assess weather conditions and aurora visibility chances

## Tech stack
React Native (Expo)
TypeScript
Expo Location API
Fetch API for HTTP requests

## External APIs
MET Norway Location Forecast API – weather data
MET Norway Sunrise API – sunrise and sunset times
NOAA Space Weather API – planetary K-index for aurora forecasting
